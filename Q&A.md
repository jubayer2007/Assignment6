1. What is the difference between var, let, and const?
Ans: The var & let is changeable but const is not changeable. Var use for only function scope but the let & const both use in all scope. By using var & let we can reassign a variable but using const it can't.

Ans: map() return data but forEach() can't return data. filter() use to find the find element.
